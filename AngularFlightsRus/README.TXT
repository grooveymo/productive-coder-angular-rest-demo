Deployment Process
-------------------

The RESTful service is supplied by the 'flightsrus' project. Take the war built by that project and deploy
to jboss.x/standalone/deployments folder.

The Angularjs app is supplied by this project. Similarly copy it's war to the destination given above.

Start jboss and navigate to:

	http://localhost:8080/angularflightsrus/FlightsRus.html

Hey presto!	



see: http://www.yearofmoo.com/2014/05/how-to-use-ngmessages-in-angularjs.html for custom validation when using angular 1.3+
